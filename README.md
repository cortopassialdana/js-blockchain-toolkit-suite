# JS Blockchain Toolkit Suite
✨ 基于JavaScript构建的全场景区块链开发工具集，覆盖钱包、加密、共识、交易、合约、P2P网络等核心区块链功能，开箱即用、无依赖、可直接二次开发。

## 项目介绍
本仓库是一套**纯JavaScript实现的轻量化区块链工具套件**，面向区块链入门学习、快速原型开发、教学演示与小型链上应用构建，所有代码原创无重复、结构清晰、注释完善，可直接部署与提交GitHub。

## 包含功能模块
1. **区块链钱包生成器**：自动生成随机私钥与链上地址，支持主流公链格式
2. **SHA256区块哈希工具**：为区块生成唯一防篡改哈希值
3. **P2P交易签名验签**：保障交易身份可信、数据不可篡改
4. **PoS权益证明模拟器**：实现区块链节点出块权选举逻辑
5. **极简区块链核心引擎**：完整可运行的迷你区块链，支持添加区块
6. **链上Token合约模拟器**：实现代币发行、转账、余额查询功能
7. **区块链合法性校验器**：检测区块是否被篡改、链结构是否有效
8. **AES链上数据加解密**：保护链上敏感隐私数据
9. **交易内存池Mempool**：待打包交易存储、排序、批量出块
10. **可信时间戳服务**：生成不可篡改的链上时间证明
11. **P2P节点发现模拟器**：模拟区块链节点网络加入与广播
12. **Gas费智能计算器**：根据交易数据自动计算Gas消耗

## 使用说明
所有代码为原生JavaScript，无需额外安装依赖，直接运行即可测试。
适合学习区块链原理、快速搭建Demo、开发小型去中心化应用（DApp）。
